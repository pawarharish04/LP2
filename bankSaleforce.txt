Step 6: Create Apex Class

In Developer Console:

👉 File → New → Apex Class

Class Name:
BankAccountSystem

Click:
👉 OK

Delete existing code and paste this:

public class BankAccountSystem {

    // Add Account
    public static void addAccount(String accId, String name, String email, Date dob, String dept) {

        BankAccount__c acc = new BankAccount__c();

        acc.Name = accId;
        acc.Emp_Name__c = name;
        acc.Email__c = email;
        acc.Birth_Date__c = dob;
        acc.Department__c = dept;

        insert acc;

        System.debug('Account Added Successfully!');
    }

    // View Accounts
    public static void viewAccounts() {

        List<BankAccount__c> accList = [
            SELECT Name, Emp_Name__c, Email__c,
                   Birth_Date__c, Department__c
            FROM BankAccount__c
        ];

        for(BankAccount__c acc : accList) {

            System.debug('Account ID: ' + acc.Name);
            System.debug('Name: ' + acc.Emp_Name__c);
            System.debug('Email: ' + acc.Email__c);
            System.debug('DOB: ' + acc.Birth_Date__c);
            System.debug('Department: ' + acc.Department__c);

            System.debug('----------------------');
        }
    }

    // Update Account
    public static void updateAccount(String accId, String newEmail) {

        BankAccount__c acc = [
            SELECT Id, Email__c
            FROM BankAccount__c
            WHERE Name = :accId
            LIMIT 1
        ];

        acc.Email__c = newEmail;

        update acc;

        System.debug('Account Updated Successfully!');
    }

    // Delete Account
    public static void deleteAccount(String accId) {

        BankAccount__c acc = [
            SELECT Id
            FROM BankAccount__c
            WHERE Name = :accId
            LIMIT 1
        ];

        delete acc;

        System.debug('Account Deleted Successfully!');
    }
}

Important Concepts for Viva
What is Salesforce?

Salesforce is a cloud-based CRM platform.

CRM = Customer Relationship Management.

What is Apex?

Apex is Salesforce’s programming language similar to Java.

Used for:

Business logic
Automation
Database operations
What is SOQL?

SOQL = Salesforce Object Query Language

Used to fetch records from Salesforce database.

Example:

SELECT Name FROM BankAccount__c
What is CRUD?
Operation	Meaning
Create	Insert
Read	Select
Update	Modify
Delete	Remove
Why __c ?

Custom objects and custom fields in Salesforce end with:

__c

Example:

BankAccount__c
Email__c
Important Viva Questions
Difference between Standard Object and Custom Object
Standard Object	Custom Object
Predefined by Salesforce	Created by user
Example: Account	Example: BankAccount
What is Execute Anonymous?

Used to run Apex code directly without creating UI.

What is System.debug()?

Used to print output in logs.

Example:

System.debug('Hello');
Common Errors + Fixes
Invalid field

Means field API name mismatch.

Check:
Fields & Relationships → API Name

Variable does not exist

Usually typo issue.

Check spelling carefully.

List has no rows for assignment

Occurs when update/delete record not found.

Example:

WHERE Name = :accId

No matching record exists.
